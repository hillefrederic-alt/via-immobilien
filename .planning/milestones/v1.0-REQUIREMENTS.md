# Requirements: Via Immobilien Website

**Defined:** 2026-01-30
**Core Value:** Vertrauen aufbauen durch authentische, ehrliche Darstellung — Besucher sollen sich verstanden fühlen und den ersten Schritt wagen (Kontaktaufnahme).

## v1 Requirements

Requirements für den initialen Launch. Alle erfolgreich geliefert.

### Legal & Compliance

- [x] **LEGAL-01**: Impressum-Seite mit allen Pflichtangaben für Immobilienmakler ✓
- [x] **LEGAL-02**: Datenschutzerklärung (DSGVO-konform) ✓
- [x] **LEGAL-03**: Cookie-Consent-Banner mit Opt-in/Opt-out ✓
- [x] **LEGAL-04**: SSL/HTTPS-Verschlüsselung (via Hosting) ✓

### Seiten & Navigation

- [x] **PAGE-01**: Startseite mit Hero-Bereich, Wertversprechen und Call-to-Action ✓
- [x] **PAGE-02**: Über mich Seite mit Foto und persönlicher Geschichte ✓
- [x] **PAGE-03**: Leistungen Seite (Verkauf und Vermietung getrennt) ✓
- [x] **PAGE-04**: So arbeite ich Seite (4-Schritte-Prozess visualisiert) ✓
- [x] **PAGE-05**: Kontakt Seite mit Formular und Kontaktdaten ✓
- [x] **PAGE-06**: FAQ Seite mit häufigen Fragen ✓
- [x] **PAGE-07**: Navigation mit allen Seiten (Header) ✓
- [x] **PAGE-08**: Footer mit Kontaktdaten, Social Links, Legal-Links ✓

### Kontakt & Erreichbarkeit

- [x] **CONTACT-01**: Kontaktformular mit Validierung (Name, E-Mail, Nachricht, Anliegen) ✓
- [x] **CONTACT-02**: Klickbare Telefonnummer (click-to-call) ✓
- [x] **CONTACT-03**: Klickbare E-Mail-Adresse (mailto) ✓
- [x] **CONTACT-04**: OpenStreetMap-Einbindung (Region/Standort) ✓

### Design & Branding

- [x] **DESIGN-01**: Responsive Design (Mobile, Tablet, Desktop) ✓
- [x] **DESIGN-02**: Branding umgesetzt (Navy Blue #142333, Cream #F5F0EA, Soft Gold #D1B787) ✓
- [x] **DESIGN-03**: Logo korrekt eingebunden ✓
- [x] **DESIGN-04**: Professionelle Typografie passend zum Branding ✓
- [x] **DESIGN-05**: Scroll-Animationen (Elemente einblenden beim Scrollen) ✓

### Trust & Social Proof

- [x] **TRUST-01**: Kundenstimmen/Testimonials-Bereich ✓
- [x] **TRUST-02**: Social Media Links (Instagram, ggf. weitere) ✓

### Performance & SEO

- [x] **PERF-01**: Ladezeit unter 3 Sekunden ✓
- [x] **PERF-02**: Lighthouse Score mindestens 90 ✓
- [x] **SEO-01**: Meta-Tags für alle Seiten (Title, Description) ✓
- [x] **SEO-02**: Open Graph Tags für Social Sharing ✓
- [x] **SEO-03**: Strukturierte Daten (LocalBusiness Schema) ✓
